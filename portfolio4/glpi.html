Ce projet comprend :

  • L’installation complète de GLPI sur un serveur
  • La configuration des collecteurs de mail pour la gestion des tickets par email
  • La rédaction de tutoriels détaillés pour chaque étape

Extrait du tutoriel d’installation

1. Prérequis

  • Serveur Linux Debian
  • Apache
  • PHP
  • MariaDB
  • Accès root et sudo

2. Installation des dépendances

# Mise à jour des paquets sudo apt update && sudo apt upgrade -y # Installation d'Apache, MySQL, PHP et extensions requises sudo apt install apache2 mariadb-server php php-{cli,common,curl,gd,imap,ldap,mysql,intl,xmlrpc,mbstring,bcmath,zip,bz2}

3. Configuration de la base de données

# Connexion à MySQL sudo mysql -u root -p # Création de la base et de l'utilisateur GLPI CREATE DATABASE glpi; CREATE USER 'glpi'@'localhost' IDENTIFIED BY 'votre_mot_de_passe'; GRANT ALL PRIVILEGES ON glpi.* TO 'glpi'@'localhost'; FLUSH PRIVILEGES;

Pour la suite, consulter le PDF complet.

Extrait : Configuration des collecteurs de mail

1. Préparation du serveur mail

Assurez-vous que votre serveur peut recevoir des emails.

2. Configuration dans GLPI

Dans GLPI, allez dans Configuration → Collecteurs de mail :

  • Ajouter un nouveau collecteur
  • Renseigner l’adresse email, le serveur IMAP/POP, le login/mot de passe
  • Définir les règles de création de ticket (projet, catégorie, priorité)
  • Tester la connexion et sauvegarder

Pour plus de détails, consulter le PDF dédié.
